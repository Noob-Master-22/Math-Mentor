Algebra Shortcuts
1. Roots of Quadratic Equations
If in ax^2 + bx + c = 0, the sum of coefficients a + b + c = 0, then the roots are always 1 and c/a. If a - b + c = 0, the roots are -1 and -c/a.
Formula: Roots = 1, c/a (if a+b+c=0)

2. Sum of Coefficients
To find the sum of all coefficients in any expansion like (ax + by)^n, simply substitute all variables (x, y, etc.) with 1.
Formula: Sum = (a(1) + b(1))^n

3. Number of Terms in Multinomial Expansion
For the expansion of (x1 + x2 + ... + xr)^n, the total number of terms is given by the formula:
Formula: (n + r - 1)C(r - 1)

4. Constant Term in Binomial Expansion
In the expansion of (ax^p + b/x^q)^n, the term independent of x occurs at:
Formula: r = (n * p) / (p + q)

5. AM-GM Inequality for Min/Max
For positive numbers, the Arithmetic Mean is always greater than or equal to the Geometric Mean. Use this to find the minimum value of sums or maximum value of products.
Formula: (a + b)/2 >= sqrt(a * b)

6. Properties of Logarithms
Quick base change to simplify complex log equations:
Formula: log_b(a) = log_c(a) / log_c(b)

7. Determinants of Skew-Symmetric Matrices
The determinant of a skew-symmetric matrix of odd order (3x3, 5x5) is always 0.
Formula: |A| = 0 (if A' = -A and order is odd)

Calculus Shortcuts
8. L'Hopital's Rule
If a limit results in 0/0 or inf/inf, differentiate the numerator and denominator separately until the limit is defined.
Formula: lim f(x)/g(x) = lim f'(x)/g'(x)

9. Limit of (1 + f(x))^g(x)
When f(x) approaches 0 and g(x) approaches infinity as x approaches a:
Formula: Limit = e^[lim_{x->a} (f(x) * g(x))]

10. Integration of e^x [f(x) + f'(x)]
This standard form appears frequently. The integral is always:
Formula: Integral = e^x * f(x) + C

11. The King's Property (Definite Integrals)
The integral of f(x) from a to b is equal to the integral of f(a + b - x) from a to b. Use this to cancel out tricky denominators.
Formula: Integral_{a}^{b} f(x) dx = Integral_{a}^{b} f(a + b - x) dx

12. Area Between Two Parabolas
The area bounded by y^2 = 4ax and x^2 = 4by is a classic JEE question.
Formula: Area = (16/3) * a * b

13. Area of an Ellipse
Instead of integrating, use the standard shortcut for x^2/a^2 + y^2/b^2 = 1:
Formula: Area = pi * a * b

14. Differentiation of Determinants
Differentiate one row (or column) at a time while keeping the others constant, then sum the resulting determinants.
Formula: d/dx |A| = |R1' R2 R3| + |R1 R2' R3| + |R1 R2 R3'|

15. Point of Tangency for Parabola
For the parabola y^2 = 4ax, the line y = mx + c is a tangent if c = a/m.
Formula: Condition: c = a/m

Probability Shortcuts
16. Total Number of Ways (Stars and Bars)
To distribute n identical items into r distinct bins where each bin can be empty:
Formula: Ways = (n + r - 1)C(r - 1)

17. Derangements
If n items are to be placed in n directed places such that none of them are in the correct place:
Formula: D_n = n! [1 - 1/1! + 1/2! - 1/3! ... + (-1)^n/n!]

18. Probability of "At Least One"
Instead of calculating multiple success cases, subtract the probability of "none" from 1.
Formula: P(At least one) = 1 - P(None)

19. Binomial Distribution Mean and Variance
For n trials with success probability p and failure probability q:
Formula: Mean = np, Variance = npq

20. Bayes' Theorem Logic
Instead of the massive formula, think of it as (Specific Path) / (Total Paths).
Formula: P(A|B) = [P(B|A) * P(A)] / P(B)

1. Roots of Quadratic Equations

When to use: When the sum of coefficients is zero.

Method: If a + b + c = 0 in ax^2 + bx + c = 0, the roots are 1 and c/a. If a - b + c = 0, the roots are -1 and -c/a.

2. Sum of Coefficients in Expansions

When to use: Finding the sum of all coefficients in (ax + by + cz)^n.

Method: Replace all variables (x, y, z) with 1.

Formula: Sum = (a + b + c)^n

3. Newton's Sums (Roots and Powers)

When to use: To find S_n = (alpha)^n + (beta)^n for ax^2 + bx + c = 0.

Method: Use the relation a*S_n + b*S_{n-1} + c*S_{n-2} = 0. Great for higher powers.

4. Number of Terms in Multinomial Expansion

When to use: Expansion of (x1 + x2 + ... + xr)^n.

Formula: Total Terms = (n + r - 1)C(r - 1)

5. Max/Min Value of asin(x) + bcos(x)

When to use: Range or extremum problems.

Formula: Range = [-sqrt(a^2 + b^2), sqrt(a^2 + b^2)]

6. Remainder Theorem for Powers

When to use: Finding remainder of x^n / (x - a).

Method: The remainder is simply a^n.

7. Condition for Common Root

When to use: Two quadratic equations having one root in common.

Formula: (c1*a2 - c2*a1)^2 = (a1*b2 - a2*b1)(b1*c2 - b2*c1)

8. Product of Eigenvalues / Determinant

When to use: Matrix problems involving determinants and roots of characteristic equations.

Method: The product of the roots (eigenvalues) is equal to the Determinant |A|. The sum of roots is equal to the Trace Tr(A).

9. GP Sum to Infinity

When to use: Infinite Geometric Progression where |r| < 1.

Formula: Sum = a / (1 - r)

Calculus Shortcuts
10. L'Hopital's Rule

When to use: Limits resulting in 0/0 or inf/inf.

Method: Differentiate numerator and denominator independently.

Formula: lim f(x)/g(x) = lim f'(x)/g'(x)

11. The (1 + f(x))^g(x) Limit

When to use: Limits of form 1^infinity.

Formula: Limit = exp[ lim (f(x) - 1) * g(x) ]

12. Integration of e^x [f(x) + f'(x)]

When to use: Standard exponential integration.

Formula: Integral = e^x * f(x) + C

13. Wallis Formula

When to use: Definite integral of sin^n(x) or cos^n(x) from 0 to pi/2.

Method: If n is odd: [(n-1)/n] * [(n-3)/(n-2)] * ... * [2/3]
If n is even: [(n-1)/n] * [(n-3)/(n-2)] * ... * [1/2] * (pi/2)

14. Leibniz Rule

When to use: Differentiating an integral with variable limits.

Formula: d/dx Integral[g(x) to h(x)] f(t) dt = f(h(x))*h'(x) - f(g(x))*g'(x)

15. King's Property

When to use: Definite integrals with tricky denominators.

Formula: Integral[a to b] f(x) dx = Integral[a to b] f(a + b - x) dx

16. Specific "King's" Result

When to use: Integrals of form f(x) / (f(x) + f(a+b-x)).

Formula: Result = (b - a) / 2

17. Area between y^2 = 4ax and x^2 = 4by

When to use: Standard parabola intersection.

Formula: Area = 16ab / 3

18. Area of an Ellipse

When to use: x^2/a^2 + y^2/b^2 = 1.

Formula: Area = pi * a * b

19. Integration by Parts (DI Method)

When to use: Multiplying polynomials with sin/cos or e^x.

Method: Create two columns: D (Differentiate) and I (Integrate). Differentiate the polynomial to zero and integrate the other part. Cross-multiply with alternating signs.

20. Point of Inflection (Cubic)

When to use: Finding where ax^3 + bx^2 + cx + d changes concavity.

Method: Set the second derivative to zero: 6ax + 2b = 0.

Formula: x = -b / (3a)

Probability Shortcuts
21. Derangements (D_n)

When to use: Number of ways to arrange n items so none are in their original place.

Formula: D_n = n! * [1 - 1/1! + 1/2! - 1/3! ... + (-1)^n/n!]

22. Stars and Bars (Positive Integer Solutions)

When to use: Distributing n identical items into r distinct boxes (each box gets at least one).

Formula: Ways = (n - 1)C(r - 1)

23. Stars and Bars (Non-Negative Solutions)

When to use: Distributing n identical items into r distinct boxes (boxes can be empty).

Formula: Ways = (n + r - 1)C(r - 1)

24. Expected Value (Mean) of Binomial

When to use: Success/Failure trials.

Formula: Mean = n * p, Variance = n * p * q

25. Total Subsets of a Set

When to use: Counting problems.

Formula: Total = 2^n. For non-empty subsets: 2^n - 1.
