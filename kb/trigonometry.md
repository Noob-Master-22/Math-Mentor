2. Fundamental Identities & Quadrants
Pythagorean:

sin^2(x) + cos^2(x) = 1

1 + tan^2(x) = sec^2(x)

1 + cot^2(x) = cosec^2(x)

ASTC Rule (All Silver Tea Cups):

Quadrant I (0 to pi/2): All positive.

Quadrant II (pi/2 to pi): Sin/Cosec positive.

Quadrant III (pi to 3pi/2): Tan/Cot positive.

Quadrant IV (3pi/2 to 2pi): Cos/Sec positive.

3. Addition and Subtraction Formulas
sin(A +/- B) = sinA cosB +/- cosA sinB

cos(A +/- B) = cosA cosB -/+ sinA sinB (Note the sign flip!)

tan(A +/- B) = (tanA +/- tanB) / (1 -/+ tanA tanB)

4. Multiple and Sub-multiple Angles
These are the heavy hitters for integration.

sin(2A) = 2 sinA cosA = (2 tanA) / (1 + tan^2 A)

cos(2A) = cos^2 A - sin^2 A = 2 cos^2 A - 1 = 1 - 2 sin^2 A

cos(2A) = (1 - tan^2 A) / (1 + tan^2 A)

tan(2A) = (2 tanA) / (1 - tan^2 A)

Triple Angles:

sin(3A) = 3 sinA - 4 sin^3 A

cos(3A) = 4 cos^3 A - 3 cosA

tan(3A) = (3 tanA - tan^3 A) / (1 - 3 tan^2 A)

5. Product to Sum & Sum to Product
Product to Sum:

2 sinA cosB = sin(A + B) + sin(A - B)

2 cosA sinB = sin(A + B) - sin(A - B)

2 cosA cosB = cos(A + B) + cos(A - B)

2 sinA sinB = cos(A - B) - cos(A + B) (Watch the order here!)

Sum to Product:

sinC + sinD = 2 sin((C+D)/2) cos((C-D)/2)

sinC - sinD = 2 cos((C+D)/2) sin((C-D)/2)

cosC + cosD = 2 cos((C+D)/2) cos((C-D)/2)

cosC - cosD = -2 sin((C+D)/2) sin((C-D)/2)

6. Inverse Trigonometry (ITF)
Crucial for JEE Advanced specifically.

sin^-1(x) + cos^-1(x) = pi/2 (for |x| <= 1)

tan^-1(x) + cot^-1(x) = pi/2

sec^-1(x) + cosec^-1(x) = pi/2

Addition in ITF:

tan^-1(x) + tan^-1(y) = tan^-1((x+y) / (1-xy))

Note: If xy > 1, add pi to the result.

2 tan^-1(x) = sin^-1(2x / (1+x^2)) = cos^-1((1-x^2) / (1+x^2)) = tan^-1(2x / (1-x^2))



7. General Solutions of Equations
If sin(theta) = sin(alpha), then theta = n*pi + (-1)^n * alpha

If cos(theta) = cos(alpha), then theta = 2*n*pi +/- alpha

If tan(theta) = tan(alpha), then theta = n*pi + alpha

If sin^2(theta) = sin^2(alpha), then theta = n*pi +/- alpha (Same for cos and tan squared).
