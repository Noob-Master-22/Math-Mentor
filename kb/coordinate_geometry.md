1. The Basics: Points and Lines
Distance Formula
The distance between P(x1, y1) and Q(x2, y2):
Distance = sqrt((x2 - x1)^2 + (y2 - y1)^2)

Section Formula
For a point dividing the line joining (x1, y1) and (x2, y2) in ratio m:n:

Internal: ((m*x2 + n*x1) / (m+n) , (m*y2 + n*y1) / (m+n))

External: ((m*x2 - n*x1) / (m-n) , (m*y2 - n*y1) / (m-n))

Area of a Triangle
With vertices (x1, y1), (x2, y2), (x3, y3):
Area = 0.5 * |x1(y2 - y3) + x2(y3 - y1) + x3(y1 - y2)|

2. The Circle
Standard Equation
Center at (0, 0) and radius r:
x^2 + y^2 = r^2

General Equation
x^2 + y^2 + 2gx + 2fy + c = 0

Center: (-g, -f)

Radius: sqrt(g^2 + f^2 - c)

Tangent Equations

At Point (x1, y1): x*x1 + y*y1 + g(x + x1) + f(y + y1) + c = 0

Condition of Tangency: For y = mx + c to touch x^2 + y^2 = a^2:
c^2 = a^2 * (1 + m^2)

3. The Parabola
Standard form: y^2 = 4ax (opens right)

Focus: (a, 0)

Directrix: x = -a

Latus Rectum Length: 4a

Parametric Coordinates: (at^2, 2at)

Tangent Equations

Point Form: y*y1 = 2a(x + x1)

Slope Form: y = mx + a/m (Crucial for "common tangent" problems!)

4. The Ellipse
Standard form: x^2/a^2 + y^2/b^2 = 1 (where a > b)

Eccentricity (e): e = sqrt(1 - b^2/a^2)

Foci: (+/- ae, 0)

Directrices: x = +/- a/e

Latus Rectum Length: 2b^2 / a

Tangent Equations

Point Form: (x*x1)/a^2 + (y*y1)/b^2 = 1

Slope Form: y = mx +/- sqrt(a^2 * m^2 + b^2)

5. The Hyperbola
Standard form: x^2/a^2 - y^2/b^2 = 1

Eccentricity (e): e = sqrt(1 + b^2/a^2)

Foci: (+/- ae, 0)

Latus Rectum Length: 2b^2 / a

Rectangular Hyperbola: xy = c^2 (Eccentricity is always sqrt(2))

Tangent Equations

Point Form: (x*x1)/a^2 - (y*y1)/b^2 = 1

Slope Form: y = mx +/- sqrt(a^2 * m^2 - b^2)
