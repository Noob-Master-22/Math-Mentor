1. Logarithms: The Triple Threat

When you see log_b(a), you must solve for three separate conditions simultaneously:

Argument must be positive: a > 0

Base must be positive: b > 0

Base cannot be one: b != 1 (This is the one students usually forget).

2. Square Roots and Even Roots

For any function f(x) inside an even root (square root, 4th root, etc.):

Constraint: f(x) >= 0

Note: If the root is in the denominator, the condition becomes strictly f(x) > 0.

Odd Roots: Functions like cuberoot(x) are defined for all real numbers (positive, negative, and zero).

3. Fractions (Rational Functions)

This is basic but lethal if ignored in complex algebraic expressions:

Constraint: Denominator != 0

Range Tip: For a linear-over-linear function like y = (ax + b) / (cx + d), the range is always All Real Numbers except a/c.

4. Trigonometric Functions

Trig functions have specific "danger zones" where they approach infinity.

Sin(x) and Cos(x): * Domain: All Real Numbers.

Range: [-1, 1]

Tan(x) and Sec(x):

Constraint: x != (2n + 1) * pi/2 (No odd multiples of 90 degrees).

Cot(x) and Csc(x):

Constraint: x != n * pi (No multiples of 180 degrees).

5. Inverse Trigonometric Functions

This is where most range errors happen. You must respect the Principal Value Branch.

Sin^-1(x) and Cos^-1(x):

Domain: [-1, 1]

Range (Sin^-1): [-pi/2, pi/2]

Range (Cos^-1): [0, pi]

Tan^-1(x) and Cot^-1(x):

Domain: All Real Numbers.

Range (Tan^-1): (-pi/2, pi/2)

Range (Cot^-1): (0, pi)

Sec^-1(x) and Csc^-1(x):

Domain: x <= -1 OR x >= 1 (Equivalent to |x| >= 1)

6. Greatest Integer Function [x] and Fractional Part {x}

[x] (Floor function): Domain is All Real Numbers, but the Range is always Integers (Z).

{x} (Fractional part): Domain is All Real Numbers, but the Range is always [0, 1).

Key Identity: x = [x] + {x}.

7. Exponential Functions

For a function like y = a^x (where a > 0):

Domain: All Real Numbers.

Range: (0, infinity). An exponential function will never be zero or negative.
